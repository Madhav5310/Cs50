# CS50's Introduction to Computer Science

All my solutions to Harvard University's CS50x: Introduction to Computer Science.

## About CS50

CS50 is an open-courseware program from Harvard University taught by David J. Malan. It introduces the intellectual enterprises of computer science and the art of programming, teaching students how to think algorithmically and solve problems efficiently.

Throughout the course, I learned core concepts including algorithms, data structures, abstraction, recursion, memory management, resource management, security, software engineering, databases, and web development.
